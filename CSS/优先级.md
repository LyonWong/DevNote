# CSS样式优先级

## 继承优先级

就近原则

- 越近继承的越高
- 直接样式优先级最高

## 选择优先级/直接样式优先级

优先级从高到低
0. 内联样式
1. id `#id`
2. 类 `.class`
3. 属性 `[type=text]`/伪类 `:hover`
4. 标签 `input`/伪元素 `:before`

- 分级别依次比较
- 同级别描述越详细（选择器数量越多）的越优先
- 继承关系越近的越优先
- 后写比先写优先

## !important
- 有`!important`的优先
- 若存在多个`!important`，按选择优先级方式计算
